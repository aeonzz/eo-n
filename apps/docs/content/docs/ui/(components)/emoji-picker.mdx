## About

The emoji picker component is built using the [Frimousse](https://frimousse.liveblocks.io/) library built by [Liveblocks](https://github.com/liveblocks).


## Usage

Import all parts and piece them together.

```tsx
import {
  EmojiPicker,
  EmojiPickerContent,
  EmojiPickerSearch,
} from "@/components/ui/emoji-picker";
```

```tsx
<EmojiPicker
  onEmojiSelect={({ emoji }) => {
    console.log(emoji);
  }}
>
  <EmojiPickerSearch />
  <EmojiPickerContent />
</EmojiPicker>
```
