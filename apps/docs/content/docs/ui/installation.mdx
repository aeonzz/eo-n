<Callout title="Note">
  Eo-N UI utilizes the shadcn cli, making it easy to install and customize
  components effortlessly.
</Callout>

<Callout title="Important" type="warn">
  This setup uses **Tailwind v4**. Make sure your project is compatible
  before proceeding.
</Callout>

<Steps>
  <Step>
    ### Setup shadcn.

    Refer to the [shadcn installation guide](https://ui.shadcn.com/docs/installation) to configure your project.

  </Step>

  <Step>
    ### Install Base UI using a package manager.

    ```package-install
    npm i @base-ui-components/react
    ```

<Callout>
  All components are in one package, and Base UI includes only what you use.
  [learn more](https://base-ui.com/react/overview/quick-start#set-up-portals)
</Callout>

  </Step>
  
  <Step>
    ### Setup portals

    Apply the ```Root``` class to your app layout.

    ```tsx title="layout.tsx"
    <body>
    // [!code highlight:3]
      <div className="Root">
        {children}
      </div>
    </body>
    ```

    Modify the ```globals.css``` file.

    ```css title="globals.css"
    @import "tailwindcss";

    @layer base {
      * {
        @apply border-border;
      }
      body {
        @apply bg-background text-foreground;
      }
      // [!code highlight:3]
      .Root {
        @apply isolate;
      }

    }
    ```

<Callout>
  Learn more about
  [portals](https://base-ui.com/react/overview/quick-start#set-up-portals).
</Callout>

  </Step>

  <Step>
    ### Configure registry

    Add the eo-n UI registry to your ```components.json``` file.

    ```json title="components.json"
    {
      "registries": {
        "@eo-n": "https://https://eo-n.vercel.app/r/{name}"
      }
    }
    ```

<Callout>
  Learn more about
  [namespaces](https://ui.shadcn.com/docs/registry/namespace).
</Callout>    

  </Step>

  <Step>
    ### ðŸŽ‰ That's it!

    You can now start adding components to your project.

    ```package-install
    npx shadcn@latest add @eo-n/button
    ```

  </Step>
  
</Steps>
